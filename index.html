Consejos de seguridad


  • Utiliza contraseñas seguras y únicas para cada cuenta que tengas

  • No compartas información personal o financiera en línea

  • No hagas clic en enlaces sospechosos o descargues archivos de fuentes desconocidas

  • Mantén tu software y antivirus actualizados

Trucos informáticos


Aquí te compartimos algunos trucos para mejorar tu productividad en la computadora:


  • Utiliza atajos de teclado para realizar tareas comunes más rápidamente

    • Algunos atajos comunes:


    1. Copiar: Ctrl + C.

    2. Cortar: Ctrl + X.

    3. Pegar: Ctrl + V.

    4. Maximizar ventana: F11o Windows del logotipo + flecha arriba.

    5. Abrir la vista de tareas: Windows del logotipo + Tab.

    6. Mostrar y ocultar el escritorio: Windows del logotipo + D.

    7. Cambiar entre aplicaciones abiertas: Alt + Tab.


  • Usa aplicaciones de organización para mantener tus tareas y proyectos en orden

  • Prueba diferentes navegadores web para encontrar el que mejor se adapte a tus necesidades

    • Algunas opciones de navegadores:


    1. Google Chrome

    2. Microsoft Edge

    3. Opera

    4. Safari


  • Realiza copias de seguridad de tus archivos importantes regularmente
